About Slidy



Slidy About How to make


Why?

Slidy is made to keep your belongings safe and help you be more organised.
Its made to make your smart home more smart.
It Reassures you everything is in the right place.


How?

Slidy started as a college Internet of Things project.

What?

The technologies

Slidy mainly uses communication through telegram with a mobile device.
This communication is made possible with telegrams BotFather which makes a bot aswell as provides with a api key
A python library called telepot handles the messaging with the bot and is used with python.

The Devices

The main device that powers Slidy is the raspberry pi.
The locking mechanisim uses a servo motor.
The weightsensing device is a load cell.
